Классы:

1. Flower
   - Описание: Класс, представляющий цветок.
   - Атрибуты:
     - id: Уникальный идентификатор цветка.
     - __name: Имя цветка.
     - __color: Цвет цветка.
     - __price: Цена цветка. 
     - __transaction_history: История транзакций.
   - Методы:
     - __init__(name="Незнакомый цветок", color="неизвестный", price=0): Конструктор для инициализации цветка.
     - __str__(): Возвращает строковое представление цветка.
     - name: Геттер для имени цветка.
     - color: Геттер для цвета цветка.
     - price: Геттер и сеттер для цены цветка (проверяет, что цена не отрицательная).
     - get_description(): Возвращает строку с описанием цветка.
     - set_name(name): Устанавливает новое имя цветка и записывает транзакцию.
     - __del__(): Метод, вызываемый при удалении объекта.
     - __add__(other): Перегружает оператор + для сложения цен двух цветков.
     - __sub__(other): Перегружает оператор - для вычисления разницы цен двух цветков.
     - __mul__(other): Перегружает оператор * для умножения цены цветка на число.
     - __truediv__(other): Перегружает оператор / для деления цены цветка на число.

2. Rose
   - Описание: Класс, представляющий розу, наследуется от класса Flower.
   - Атрибуты:
     - thorn_length: Длина шипов у розы.
   - Методы:
     - __init__(color, price, thorn_length): Конструктор для инициализации розы с заданными параметрами.
     - get_description(): Возвращает строку с описанием розы, включая длину шипов.

3. Tulip
   - Описание: Класс, представляющий тюльпан, наследуется от класса Flower.
   - Атрибуты:
     - petal_count: Количество лепестков у тюльпана.
   - Методы:
     - __init__(color, price, petal_count): Конструктор для инициализации тюльпана с заданными параметрами.
     - get_description(): Возвращает строку с описанием тюльпана, включая количество лепестков.

4. Bouquet
   - Описание: Класс, представляющий букет цветов.
   - Атрибуты:
     - flowers: Список для хранения цветов в букете.
     - __transaction_history: История транзакций.
   - Методы:
     - __init__(): Конструктор для инициализации букета.
     - add_flower(flower): Добавляет цветок в букет и записывает транзакцию.
     - get_flowers(): Возвращает список цветов в букете.
     - total_price(): Возвращает общую цену всех цветов в букете.
     - __str__(): Возвращает строковое представление букета.
     - serialize(): Сериализует букет в файл.
     - deserialize(): Десериализует букет из файла.
